Messaging for [Bajo](https://github.com/ardhi/bajo)

## Installation

Goto your ```{project-dir}``` and type:

```bash
$ npm install masohi bajo-queue
```

Now open your ```{data-dir}/config/.plugins``` and add ```masohi``` and ```bajo-queue```
